Each version should be denser — packing more information into fewer words.\n\nText: [TEXT]\n\nVersion 1 (longest):\nVersion 2:\nVersion 3:\nVersion 4:\nVersion 5 (shortest, most dense):
# Tree of Thought

# Socratic Method

# SCAMPER Framework

# First Principles

# Persona Switching

# Iterative Refinement

# MECE Analysis

# Audience Adaptation

# Contrarian Thinking
Take the contrarian position on: [CLAIM/IDEA]\n\nPresent the strongest possible argument against the mainstream view. Include evidence, reasoning, and address likely counterarguments. End with an honest assessment of how strong the contrarian case actually is.
